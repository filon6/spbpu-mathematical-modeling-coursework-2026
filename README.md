## Метод решения

Для нахождения точек бифуркации Андронова-Хопфа используются условия

trace(J) = 0  

det(J) > 0  

где J — матрица Якоби системы.

### Алгоритм решения

1. Рассматривается система стационарных уравнений.
2. Первое уравнение умножается на \(p_4\) и вычитается из второго.
3. Получается выражение для переменной \(x_1\):

x1 = ((1 + p5) * x2) / p4

4. Из первого уравнения выражается величина \(p_2 E(x_2)\).
5. Полученные выражения подставляются в условие

trace(J) = 0

6. После преобразований получается **квадратное уравнение относительно**

y = 1 + p5

7. Для различных значений \(x_2\) и \(p_4\) находятся допустимые корни.

Расчёты выполнялись с использованием Python.

## Вторая консультация

После отправки таблиц преподаватель отметил, что найденные значения удовлетворяют **необходимому условию бифуркации**

Однако это условие **не является достаточным**.

Для подтверждения бифуркации необходимо дополнительно проверить условие

det(J) > 0

После добавления этих проверок можно приступать к оформлению итогового отчёта.
